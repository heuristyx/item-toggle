# ItemToggle
is a mod that lets you easily enable or disable items within the system provided by the Celeste (Open World) randomizer. This includes interactable objects, keys, dash directions, climb directions and the ability to crouch.

## Installation
Download the latest version from the releases page and drop it into your Everest `Mods` folder. The mod is dependent on Celeste (Open World) v1.1.0+.

## Usage
Items can be turned on or off in the item toggle menu, opened with `Z` (keyboard) or `L3 (Press Left Stick)` (controller) by default. Bindings can be changed in the settings.
